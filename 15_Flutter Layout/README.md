# Flutter Layout

- Layout merupakan suatu widget yang digunakan untuk mengatur tata letak widget lainnya. Terdapat 2 jenis widget untuk mengatur layout yaitu Single Child widget dan Multiple Child widget.
- Single Child widget merupakan widget yang hanya dapat memiliki satu widget didalamnya yang diberi nama property '' untuk memasukchildan widget. Beberapa widget single child diantaranya adalah 'Container' untuk 'Center' untuk widget yang diletakkan ditengah, 'sizmembungkus widget dalam bentuk box yang bisa disesuaikan,eBox untuk widget yang lebih sederhana.'
- Multiple Child yang dwidget merupakan widget yang dapat memiliki beberapa widget didalamnyaiantaranya adalah 'Column' untuk mengatur widget horizontal, 'Row' untuk mengatur widget secara vertikal, 'List' untuk widget dalam bentuk list yang dapat discroll, 'GridView' untuk widget dalam bentuk galeri.
